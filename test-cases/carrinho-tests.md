# Casos de Teste - Carrinho de Compras

## CT-003: Adicionar produto ao carrinho

**Descrição:** Verificar se o usuário consegue adicionar um produto ao carrinho  
**Pré-condição:** Usuário logado na página de produtos

| Passo | Ação | Resultado Esperado |
|-------|------|--------------------|
| 1 | Escolher um produto qualquer | Produto visível na tela |
| 2 | Clicar em "Add to cart" | Botão muda para "Remove" |
| 3 | Verificar o ícone do carrinho | Ícone mostra o número 1 |
